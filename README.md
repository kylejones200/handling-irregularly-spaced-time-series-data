## Business context

Most time series methods assume regular intervals. Real-world data often breaks this rule. Financial trades, social media posts, seismic events --- these arrive when they happen, not on schedule.

Irregular time series have inconsistent time gaps between observations. This breaks the assumptions behind many standard models.

- Uneven spacing between timestamps - Dense bursts followed by long gaps - Missing seasonal or trend structure
